## 2. Project Summary
Our idea targets the IRP (Individual Retirement Pension) market, which has historically shown low returns due to information asymmetry. By integrating financial data, we aim to provide transparent comparisons of returns and fees, while leveraging AI to deliver personalized portfolio recommendations for each user.
